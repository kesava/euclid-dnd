# euclid

A simple exercise to learn `loop` and `recur` by implementing Euclid's algorithm of finding set of largest same-sized squares into which a reactangle can be divided.


## Usage
```clojure
(biggestSq 20 20)
;; 20
(biggestSq 25 20)
;; 5
(biggestSq 23 17)
;; 1
(biggestSq 230 170)
;; 10
```
